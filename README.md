# Cancer Prediction using Naïve Bayes

## Project Overview
This project predicts whether a tumor is malignant or benign using machine learning techniques.
The model is built using the Naïve Bayes algorithm on the Breast Cancer dataset.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
